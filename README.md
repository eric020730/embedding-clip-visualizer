# Embedding & CLIP 視覺化工具

用 3D 互動空間理解 AI 的 Embedding 和 CLIP 圖像檢索概念。

## ✨ 功能特色

- 🎨 **Embedding 空間視覺化** - 展示文字和圖像在向量空間中的分布
- 🔍 **CLIP 搜尋演示** - 展示文字查詢如何找到相似圖像
- 🎮 **3D 互動控制** - 拖曳旋轉、按鈕縮放、重置視角
- 📱 **響應式設計** - 支援桌面、平板、手機
- ⚡ **高效能渲染** - 使用 Canvas 2D 和高 DPI 支援

## 📖 使用說明

### 控制方式
- **拖曳滑鼠** - 旋轉 3D 空間
- **+ / - 按鈕** - 放大/縮小視圖
- **重置視角** - 回到預設視角
- **切換標籤** - 查看不同模式

### Embedding 模式
展示不同類別的資料點（動物、水果、交通工具）如何在向量空間中自然聚集。

### CLIP 模式
展示文字查詢「可愛的小貓」如何通過向量相似度找到最匹配的圖像。
